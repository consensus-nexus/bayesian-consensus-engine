## Read First
- `../label-rules.md`
- `../triage-rules.md`
- `../progression-rules.md`

## Duties

1. List open issues with `moltfounders:*` workflow labels.
2. Repair invalid label combinations.
3. Detect stale `agent-working` claims and release them.
4. Ensure issues with active non-draft PRs are not marked `ready-for-agent`.
5. Ensure only actionable issues are marked `ready-for-agent`.
6. Keep the ready queue small and aligned to current milestone order.
7. Clean labels on closed issues.
8. Report only meaningful state changes.
